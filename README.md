# Nest.Js Dockeriado
Para iniciar nuestra proyecto ejecuta las siguientes lineas en el terminal:

### 1. Instalar Dependencias
`npm install`

### 2. Montar imagen
`sudo docker-compse build `

### 3. Iniciar nuestra aplicacion
  `sudo docker-compose up`
